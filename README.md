<h2>üõí Office Syndrome E-commerce Platform</h2>
<p>A full-stack e-commerce application built with the MERN stack (MongoDB, Express.js, React, Node.js) specifically designed for products and resources that alleviate <b>Office Syndrome</b> symptoms.

This platform allows users to browse ergonomic products, view health resources, manage their shopping cart, and securely complete purchases. The entire application is designed for scalability and is deployed using Vercel.</p>

<h2>‚ú® Features</h2>
<h3>General E-commerce</h3>
<ul>
<li>Product Catalog: Browse a comprehensive list of office syndrome-relief products (e.g., ergonomic chairs, standing desks, wrist rests, specialized exercise equipment).</li>

<li>Product Details: View detailed product descriptions, specifications, pricing, stock status, and customer reviews/ratings.</li>

<li>Search & Filter: Search for products by name and filter by category (e.g., Back Care, Eye Strain, Wrist Support) and price.</li>

<li>Shopping Cart: Add, remove, and update product quantities in the cart.</li>

<li>Order History: Users can view details and status of past orders.</li></ul>

<h2>üõ†Ô∏è Tech Stack</h2>
<h3>Backend & Database (Serverless on Vercel)</h3>

<ul>
<li>MongoDB: NoSQL database for flexible and scalable product and user data storage (using MongoDB Atlas for cloud hosting).</li>

<li>Node.js: JavaScript runtime environment.</li></ul>

<h3>Frontend (Hosted on Vercel)</h3>
<ul>
<li>React: JavaScript library for building the user interface.</li>

<li>React Router: For client-side routing.</li>

<li>State Management: (e.g., Redux, Redux Toolkit, or React Context) for predictable state management.</li>

<li>Styling: (e.g., Tailwind CSS, Material-UI, or Styled Components).</li>
